# DASC41103Project2
 - In order to run the project, install the repo to your computer
 - Unzip folder into whichever directory you prefer
 - Open the file `DASC41103Project2Code.ipynb` into either VSCode or Google Colab
    - If running in VSCode, no edits are needing, simply press "Run All"
    - If running in Colab, uncomment the code chunk under the heading "In Google Colab" and comment the code chunk under the heading "In VS Code" and then press "Run All"
       - When prompted by the same code chunk under "In Google Colab", select the "Choose Files" button and shift click the two files: `project_adult.csv` and `project_validation_inputs.csv`
- _note: the hyperparameter code chunk may take several minutes to run_
